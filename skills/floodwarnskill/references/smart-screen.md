# 智慧大屏功能与状态

## 智慧大屏管理

菜单路径：**智慧大屏 → 智慧大屏管理**。

列表显示监控线路名称、摄像头数量、传感器数量、创建时间和最后修改时间。页面提供 **新增大屏**、**新增大屏2**、**查看** 和 **删除**。

这些入口不是同一成熟度：

- **新增大屏**：地图型监控线路编辑，存在实际新增接口。
- **删除**：存在实际删除接口，确认后从列表移除记录。
- **新增大屏2**：背景图画布原型，保存接口被注释。
- **查看**：当前进入 `viewScreen` 原型页并加载 mock 数据，不是所选记录的真实实时大屏。

## 新增地图型大屏

1. 点击 **新增大屏**，进入地图编辑页。
2. 在左侧输入监控线路名称。
3. 在地图目标位置点击右键，添加摄像头标记。
4. 摄像头标记可以拖动调整位置。
5. 在摄像头列表点击编辑图标，填写“摄像头名称”和“视频流地址”；也可以删除标记。
6. 点击 **保存**。成功后页面创建监控线路并关闭当前标签。

保存前必须同时满足：

- 监控线路名称非空。
- 至少添加一个摄像头。
- 每个摄像头的视频流地址非空。

保存时页面同时记录地图中心点和缩放级别。地图加载失败时，应刷新页面后重试。

## 删除大屏

1. 在列表中找到目标监控线路。
2. 点击 **删除**。
3. 在确认框确认删除。

删除会调用实际删除接口。当前前端成功提示文字为“保存成功”，但其实际操作含义是删除完成并刷新列表。

## 新增大屏2：背景图画布原型

该入口用于在背景图上放置摄像头图标并预览布局：

1. 点击 **新增大屏2**。
2. 点击 **添加背景图**；背景图片不能超过 6MB。
3. 点击 **添加摄像头**，从列表选择设备，可设置默认摄像头。
4. 在画布上点击放置摄像头；可切换预览和编辑模式，也可删除已放置设备。
5. 点击 **保存** 时页面会显示“保存成功”。

重要限制：当前保存接口代码已被注释，点击保存不会把布局持久化到后端。回答用户时必须称其为原型/演示功能，不能承诺刷新或重新进入后仍保留数据。

## 查看大屏：mock 页面

列表中的 **查看** 当前进入“智慧防汛监控平台”原型页。页面展示河道水情、水库水情、实时雨量、气象信息、容库曲线、摄像头数量和在线率，并支持摄像头图层和全屏。

这些水情、雨量、气象、曲线和摄像头数据来自前端 `mockData`，不是根据列表所选大屏或实时接口加载。不得：

- 把页面数值回答为真实监测数据。
- 根据页面状态判断真实设备在线率或预警状态。
- 宣称“查看”已加载刚保存的监控线路布局。

若用户需要正式实时大屏，应说明当前前端尚未完成该查看页与真实数据、已保存大屏记录的接入。
