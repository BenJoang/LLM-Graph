# AI 训练功能与操作

## 视频处理

菜单路径：**AI 训练 → 视频处理**。

用途：把视频按固定秒数间隔转换成图片，下载图片或加入图片集。

操作步骤：

1. 在“视频来源”选择 **本地上传**，或打开 **视频库** 选择已有视频。
2. 本地视频只支持 MP4、AVI、MOV、FLV、WMV，单个文件不能超过 100MB。
3. 在“视频处理间隔”输入正整数，单位为秒。
4. 点击 **开始处理**。预计一次生成较多图片时，页面会要求二次确认。
5. 在图片列表勾选结果；可以 **全选**、**下载**，或 **上传到图片集**。

注意事项：

- 视频库支持选择和删除视频；删除前会提示“删除后无法恢复”。
- 没有选择视频、处理间隔无效或没有勾选图片时，相关操作不会执行。

## 图库管理

菜单路径：**AI 训练 → 图库管理**。

页面按当前统计显示四类集合：**图片集、标注集、数据集、尺寸集**。视频库不在此页显示。

常用操作：

1. 点击顶部集合卡片切换集合。
2. 点击文件夹进入下一级；点击面包屑返回上级或根目录。
3. 图片集和尺寸集提供 **上传图片**，支持 JPG、JPEG、PNG、WebP，可多选；不支持的文件会被过滤。
4. 勾选图片后点击 **删除图片**；删除会直接影响当前集合。
5. 使用图片区域的右键菜单新建文件夹、重命名、移动图片或删除文件夹。

文件夹规则：

- 文件夹名称不能为空，同一级不能重名。
- 取消尚未确认的新建文件夹时，该临时文件夹会被移除。
- 删除文件夹会同时删除其中图片，且页面明确提示无法恢复。
- 标注集不开放文件夹管理；图片集、数据集和尺寸集可使用文件夹管理。

## 图片标注

菜单路径：**AI 训练 → 图片标注**。

数据源包括 **图片集、标注集、尺寸集**。打开数据源后，在抽屉中进入目标文件夹、勾选图片并点击 **导入图片**。

### 从图片集创建标注

1. 选择 **图片集** 并导入图片。
2. 选择 **多边形**，在画布上依次点击顶点并完成区域。
3. 在“多边形设置”选择区域类型：**水体**会自动使用名称 `flood`；**其他**需要填写名称。
4. 如需尺度信息，选择 **标尺** 绘制线段，并输入实际长度，单位为米且必须为正数。
5. 点击 **添加到标注集** 保存。

多边形名称不能重复。切换图片时如果存在未完成编辑，页面会提示切换将丢弃内容。

### 修改标注集

1. 选择 **标注集** 并导入已有标注图片。
2. 选择列表中的多边形或标尺进行检查、调整或删除。
3. 点击 **保存标注集**。页面会反馈成功和失败的图片数量。

### 制作尺寸集

1. 选择 **尺寸集** 并导入图片。
2. 使用 **多边形** 标出目标区域，使用 **点标注** 添加坐标点。
3. 点名称必须是两个由英文或中文逗号分隔的数字，例如 `123,456`。
4. 点击 **添加关联关系**，选择一个尚未关联的多边形和恰好四个尚未关联的点。
5. 确认关联后点击 **保存尺寸集**。

建立关系前必须至少存在一个多边形和四个标注点；一个多边形或标注点不能重复用于其他关系。

## 图片质量评价

菜单路径：**AI 训练 → 图片质量评价**。

1. 从左侧选择某个用户的标注集。
2. 在文件列表选择待审核图片，检查画布上的多边形和标尺。
3. 如修改了标注，页面会先保存修改后的标注数据。
4. 点击 **审核通过** 或 **审核不通过**。

审核不通过的图片不会提交到用户数据集。当前没有待审核图片时，操作按钮不可用。

## 训练模型

菜单路径：**AI 训练 → 训练模型**。

### 创建训练任务

1. 点击 **+训练模型**。
2. 选择一个或多个数据集，并选择模型类型。
3. 模型名称可留空，由系统自动生成；也可选择已有输出模型作为输入模型。
4. 按需要调整参数，点击 **开始训练**。

前端校验范围：

| 参数 | 允许范围 |
| --- | --- |
| Freeze_Epoch | 0–80 的整数 |
| UnFreeze_Epoch | 100–200 的整数 |
| 数据集处理 train_percent | 0–1，最多 1 位小数 |
| Freeze_batch_size | 2–16 的整数 |
| UnFreeze_batch_size | 2–8 的整数 |
| num_workers | 0–4 的整数 |

其他可选项包括 downsample_factor 8/16、fp16、focal_loss、optimizer_type（adam/sgd）和 lr_decay_type（step/cos）。

### 查看和管理任务

- 训练期间页面显示进度，并暂时禁止再创建训练任务。
- 点击 **中止训练** 前会二次确认；中止后不可恢复。
- 历史列表显示输入模型、输出模型、训练耗时、创建时间、失败原因和状态。
- 点击 **查看** 只能查看参数，不能从查看窗口重新提交。
- 点击 **删除** 会永久删除对应训练历史，操作前会二次确认。
