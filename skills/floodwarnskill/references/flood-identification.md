# 山洪识别、摄像头与传感器操作

## 图片识别

菜单路径：**山洪识别 → 图片识别**。

1. 点击 **图片集** 或 **数据集**，在抽屉中进入文件夹并勾选图片。
2. 点击 **导入图片**；重复图片不会再次导入。
3. 选择“输出图片模式”和“权重模型”。
4. 点击 **开始识别**。
5. 在识别结果中勾选图片并点击 **下载**。单张直接下载，多张打包为 ZIP。

没有选择图片时不能开始识别；识别完成但没有有效区域时，页面会给出提示而不生成结果图片。

## 视频识别

菜单路径：**山洪识别 → 视频识别**。

1. 点击 **上传本地视频**，选择一个视频。
2. 支持 MP4、AVI、MOV、FLV、WMV，单个视频不能超过 100MB，一次只能上传一个。
3. 选择“识别模式”和“权重模型”。
4. 点击 **开始识别**，等待进度达到 100%。
5. 在处理后的视频区域预览结果，点击 **下载** 导出。

尺寸 JSON：

- 页面允许上传一个尺寸 JSON，必须是 JSON 对象、扩展名为 `.json`，大小不能超过 10MB。
- 当前前端只解析并显示该文件状态，启动识别时实际提交的参数只有视频文件 ID、识别模式和权重模型。
- 因此不得承诺尺寸 JSON 会影响当前视频识别结果。

如果自动下载失败，页面可能复制视频链接到剪贴板，或提示在浏览器中手动下载。进度查询失败时需要检查网络连接后重新发起任务。

## 项目摄像头直播与回放

菜单路径：**山洪识别 → 摄像头识别**。

1. 在左侧项目摄像头列表选择设备；离线设备带有“离线”标记，不能播放。
2. 查看实时画面时点击 **查看直播**。
3. 查看历史画面时先选择“回放开始时间”，再点击 **查看回放**。
4. 避免连续快速切换摄像头；页面设置了约 1.2 秒的切换保护。
5. 播放页面提示“不要切换清晰度”。

状态说明：

- “显示识别区域”“显示识别面积”“显示流速”控件目前没有进入播放器请求或处理逻辑。
- “已选择/选择标注”可以打开尺寸集抽屉，但所选标注目前也没有进入直播或回放逻辑。
- 只能将该页面回答为直播/回放查看功能，不能宣称已进行实时山洪识别、面积计算或流速计算。

常见提示包括：请先选择摄像头、请先选择回放开始时间、设备离线、切换过于频繁、直播或回放加载失败。

## WVP 摄像头

菜单路径：**山洪识别 → WVP摄像头**。

### 连接和筛选

1. 填写 WVP 地址、用户名和密码，三项缺一不可。
2. 按需勾选 **仅在线**，点击 **连接并拉取通道**。
3. 使用搜索框按通道名称或编码筛选，也可以点击 **刷新列表**。
4. 左侧列表显示通道编码及在线/离线状态。

### 分屏播放

1. 选择 **1分屏、4分屏或9分屏**。
2. 点击目标窗格，使其成为“当前窗格”。
3. 在左侧选择通道，点击 **开始播放**，把通道放入当前窗格。
4. 也可以点击 **自动填充分屏**，按通道顺序填充所有可见窗格。
5. 使用 **清空窗格** 移除当前窗格的播放内容；点击窗格右上角进行全屏或退出全屏。

播放器默认优先使用主流 `ws_flv`，失败后会尝试其他可用的 ws_flv、flv、hls 或 fmp4 地址。通道没有可播放地址、浏览器拦截全屏、音频编码不兼容且无降级地址时，页面会明确提示。

## 传感器数据

菜单路径：**山洪识别 → 传感器数据**。

1. 在左侧选择传感器；离线设备不能查看数据。
2. 选择开始和结束时间。快捷范围包括最近 7 天、最近 15 天和最近 1 个月。
3. 点击折线图按钮查看趋势，点击列表按钮查看明细。
4. 列表根据设备数据动态显示指标名称和单位，可切换每页条数或页码。

选择设备或时间范围后页面会重新获取数据。没有选择传感器、设备离线、没有数据或请求失败时，分别会显示对应提示；不要编造缺失数据或单位。
