# agenttool

## DESCRIPTION
启动一个独立的子 agent 完成复杂、多步骤、需要调用多个工具的任务。任务工作量明显大时可调用。
## PROMPT
当任务需要独立调查、跨多个文件检索、连续调用多个工具、或需要把上下文消耗隔离出去时，使用本工具。
调用时必须传入 prompt 字段。prompt 必须是完整任务说明，因为子 agent 是无状态的，看不到主 agent 的完整对话。
prompt 应包含：
- 任务目标
- 相关路径
- 需要使用的信息来源
- 期望返回格式
- 禁止编造结果
子 agent 的返回结果不会直接展示给用户，主 agent 需要根据结果再组织最终回答。

必须使用agenttool完成的任务：
1. 对某个传入的图片进行分析，需要使用agenttool拉起imageread完成，把结果返回给主agent

不要在以下情况使用 agenttool：
1. 少数普通工具调用就能完成任务。
