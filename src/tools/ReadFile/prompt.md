# read_file

## DESCRIPTION

读取文件中的正文文本，支持".docx"文件和".py", ".ts", ".tsx", ".js", ".jsx", ".json", ".md", ".txt"文件的读取。

## PROMPT
当用户需要读取、检查、总结或分析.docx文件和".py", ".ts", ".tsx", ".js", ".jsx", ".json", ".md", ".txt"文件正文内容时，使用这个工具。
这个工具读取docx时只提取普通正文段落，不保留字体、字号、颜色、分页、图片、页眉、页脚、批注或表格内容。
对于.py文件的返回，工具返回的每一行可能带有行号前缀，行号仅用于定位，不属于文件内容。在解释代码时可以引用行号；在复写代码时不要包含行号。
不要把它用于 `.doc`、PDF、图片、扫描件，或需要严格版式还原的任务。
在文档未读取完之前，truncated会返回True，请确保文档被读取完整时再做最终回答
